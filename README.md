# TUCSA · Seguimiento de Clientes (Demo)

App móvil de seguimiento comercial para **TUCSA – Transporte de Personal** (corredor Saltillo – Ramos Arizpe – Derramadero).

Demo de una sola página (HTML/JS, sin backend). Pensada para usarse desde el celular: pipeline de prospectos, embudo de ventas, agenda de seguimiento, llamada/WhatsApp/correo con un toque, e historial de actividades por cuenta.

## Ver el demo

👉 **https://ricardolopezstrozzi.github.io/tucsa-crm-demo/**

> Los datos visibles son una base inicial de prospección (empresas industriales del corredor). En modo demo, los cambios se guardan en el navegador del dispositivo.

## Conexión con Google Sheets

La app puede leer y escribir en vivo sobre un Google Sheet (Google Apps Script). Las instrucciones y el código están dentro de la propia app, en la pestaña **Ajustes → Google Sheets**.
